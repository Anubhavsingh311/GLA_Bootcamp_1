# Pen vs Book — Image Classifier

This is an AI model that can look at an image and tell you whether it sees a **Pen** or a **Book**.

It was trained using Google's Teachable Machine by showing it nearly 1,000 photos of each object.

---

## Try It (No Download Needed)

Click the link below and allow camera access when prompted. Hold up a pen or a book in front of your camera and watch the AI classify it in real time.

**[Test the model in your browser →](https://teachablemachine.withgoogle.com/models/NACXyJUzQ/)**

Works on any device with a camera. No sign-in, no install, nothing to download.

---

## Model Details

| Detail | Value |
|---|---|
| Classes | Pen, Book |
| Training samples | ~974 (Pen), ~937 (Book) |
| Epochs | 4000 |
| Batch size | 16 |
| Learning rate | 0.00001 |
| Base architecture | MobileNet (transfer learning) |
| Framework | TensorFlow.js via Teachable Machine |
